Connect-four game



----- Setup -----

1. Clone the repository (git clone <repo URL>)
2. Open repo folder, and cd <repo location> 




---------- Language choice, features ----------

Language Choice:

1. Language chosen:    Python
2. Reason:             Python is the language I use for my personal projects and thesis, and it is much simpler for readability and rapid development. 


Features:

1. The user adds their choice of grid size (rows, columns) and tokens required for winning.
2. Two user inputs (for two players).
3. Win detected horisontally, vertically, and diagonally.
4. Error handling (e.g. invalid input, draw, full column)




------- Running on Mac/Linux -------

--> How to run the game and tests:

- Terminal, by navigating to main.py's location and writing the following prompt: python3 main.py (for running the actual game) and python3 -m pytest tests/ for running the tests.

- Code editor, by simply opening main.py or tests and running them.
